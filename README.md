# Sales Data Analysis Using Python

This project analyzes sales data to understand revenue trends, product performance, and category-wise contribution.

## Project Tasks
- Created a dataset with 60 sales records
- Calculated total sales per order
- Performed product-wise and category-wise analysis
- Analyzed monthly sales trends
- Exported final dataset as CSV

## Key Insights
- High-performing products were identified using total revenue
- Electronics and Accessories categories were compared
- Monthly revenue trends were analyzed to observe sales patterns

- Python
- Google Colab
